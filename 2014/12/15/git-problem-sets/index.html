Git Problem Sets 

Command: git mv
Show: fatal: destination exists
To solve: git mv --force
Describe: 当重命名时，如果只是重命名文件名称的大小写，需要加上 --force 参数

Command: git reset, git clean
Show:
To solve: git reset --hard && git clean -xdf
Describe: 多余的文件且没被追踪，需要删除
